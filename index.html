Tales of the Clans at Highstones

Tales of the Clans at Highstones, abbreviated to TOTCAH, is a Warrior Cats fanfiction; it is an AU set in the time the Warrior Code was being created. It is an animation series posted on Scratch (scratch.mit.edu).

It has five seasons, each following different cats and territories.
